## Deploy to GitHub Pages
1. Create a **public** GitHub repository
2. Upload all files (preserving folder structure)
3. Settings → Pages → Deploy from `main` branch `/root`
4. Site live at `https://yourusername.github.io/repo-name/`

## Customisation
- **Contact details**: Edit `SITE_CONFIG` in `js/main.js`
- **Colors**: Edit CSS variables in `css/style.css` `:root` block
- **Formspree**: Replace `YOUR_FORM_ID` in `js/main.js` + all HTML files
